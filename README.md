# Omnifood — Never cook again

A premium AI-powered food subscription that will make you eat healthy again, 365 days per year. Tailored to your personal tastes and nutritional needs.

## 🚀 Project Overview
This project is a high-fidelity, responsive landing page built as the final capstone for Jonas Schmedtmann's **"Build Responsive Real-World Websites with HTML and CSS"** course. It demonstrates modern front-end development techniques, focusing on user experience, performance, and responsive design.

## 🛠️ Tech Stack & Features
- **Semantic HTML5:** Structured for SEO and accessibility.
- **Modern CSS (Grid & Flexbox):** Used for complex layouts and precise alignment.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop using a "mobile-first" approach with custom media queries.
- **JavaScript (Vanilla):** - Implementation of sticky navigation using the **Intersection Observer API**.
  - Smooth scrolling behavior for internal links.
  - Mobile navigation toggle functionality.
- **Web Manifest:** Basic PWA support for icon management.
- **Third-Party Libraries:** - [Ionicons](https://ionic.io/icons) for scalable vector icons.
  - [Google Fonts (Rubik)](https://fonts.google.com/specimen/Rubik) for typography.
  - [Smoothscroll Polyfill](https://github.com/iamdustan/smoothscroll) for cross-browser compatibility.

## 📂 Project Structure
- `index.html`: Main structure of the landing page.
- `css/`:
  - `general.css`: Reusable components, typography, and color systems.
  - `style.css`: Section-specific layouts and styling.
  - `queries.css`: Media queries for responsiveness.
- `js/script.js`: Interactive elements and UI logic.
- `img/`: Optimized images for the hero section, meals, and gallery.
